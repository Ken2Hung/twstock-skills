### 其他決策

- **基本面用同產業百分位**（`TaiwanStockInfo.industry_category`）+ 二階改善，非硬門檻——避免砍成長股；硬門檻屬 L3 使用者明訂條件。
- **缺面向降權重**：以剩餘面向權重重新正規化（如缺籌碼 → 基本/技術各 50%），不以預設值填充。
- **v1.0 long-only**：信號「賣出/觀望」= 不建議買進/退出，非做空。故 止損 ≤ 現價 ≤ 目標 在此假設下成立、寫回 spec。做空方向反轉列 future。此脈絡承接自 Change 3（add-screening-skill）design.md 之延後記錄。
- **目標價/止損語意修正（承你的修正）**：價位一律**市場價**（供觸價判定），成本**不入價位**；成本只在**預期報酬率**計算時扣來回（買 0.1425%+賣 0.4425%），引用 `tw-market-rules.md`。這修正了 Change 3 的 Req3（本 change 以 MODIFIED 更新該 requirement）。

### 第二輪裁決修訂（explore 回合 2）

- **相對量正規化（1c）**：法人淨買超與融資／融券變化 MUST 以**佔同期成交量比**計，不用絕對張數/金額，避免系統性偏向大型股。
  - 法人 metric 提案：**淨買超佔同期成交量比**（擇「佔成交量比」而非「產業百分位」——前者 per-stock 自足、不需足量同業樣本，且與融資 metric 一致）。
- **融資「大增」量化（1a，提案待你裁決）**：擇 **`增量佔同期成交量比`**（非「增幅 %」——增幅%對低基期小型股雜訊大；佔成交量比正規化到流動性，且與 1c 一致）。
  - **提案門檻 Y = 10%**（5 日融資淨增量 ÷ 同期成交量 > 10% 視為「大增」→ 負向）。此為起始經驗值，**v1.0 回測後可調**。
- **已知限制註記（1b）**：`scoring-model.md` 限制段落 MUST 註明「融資增=負」為**不分股價位置**的 v1 簡化；低檔融資增常為反彈確認，**v-next 結合位置資訊**。
- **窗口統一（2）**：籌碼/量價類回看窗口統一 **5 日**；均線多頭排列之 5/20/60 為指標結構定義（非回看窗口）；乖離 overlay 基準亦對齊短期。
- **月營收窗口定死（3）**：百分位近 **6 月**、二階改善近 **3 月**（移除「3–6 月」區間寫法）。
- **農曆年在地規則（4）**：1、2 月營收 YoY MUST 合併計算（消除農曆年落點失真），寫入 `scoring-model.md` 計算規則，並於 `tw-market-rules.md` 補一行交叉引用。
