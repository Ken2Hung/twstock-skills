# twstock-reviewing-portfolio

## 1. 觸發時機

當使用者想檢視**已持有**股票的健康狀況時觸發，例如：
- 「**幫我健檢持股** 2330、2454」
- 「我手上的 2317 **還能抱嗎**」
- 「**檢查我的持股**有沒有警訊」

找**新標的**（選股）屬 `twstock-screening-stocks`；單檔純資料查詢屬 `twstock-module`。

## 2. 能力總覽（編排流程）

L3 **編排型**，四階段：

```
解析持股清單 → 委派 twstock-module 取得資料 → 掃描警訊 → 輸出健檢看板
```

**分層鐵律（紅線 #2）**：本場景 SHALL NOT 直接呼叫 FinMind／yfinance／TWSE／TPEx 或任何資料源 API；**凡需任何台股資料，一律委派 `twstock-module`**。

## 3. 輸入：持股清單解析

把使用者輸入解析為一份持股清單（股票代碼，一至多檔；數量/成本非必要）。逐檔委派 `twstock-module` 取得健檢所需資料。

## 4. 委派取數

每檔持股所需資料 → 呼叫 `twstock-module`：`institutional`（法人）、`margin`（融資融券）、`financial`／`revenue`（財報/營收）、`daily`（價量/漲跌停）、`shareholding`（大戶持股）。**禁止**在本場景直接碰 API。回傳含 `data_gaps`，本場景原樣尊重、不補值。

## 5. 警訊掃描

沿用 [`scoring-model.md`](../twstock-screening-stocks/references/scoring-model.md) 指標與 [`tw-market-rules.md`](../twstock-screening-stocks/references/tw-market-rules.md) 規則，至少掃：

- **法人動向**：外資/投信連續**賣超**（回看窗口見 scoring-model）
- **籌碼**：融資餘額**大增**（散戶槓桿過熱）、大戶持股比下滑
- **財報**：EPS／月營收**轉弱**（YoY 走弱，1+2 月合併計）
- **市場**：觸及漲跌停（±10%）、列入 TWSE **處置名單**

指標算法與成本/處置規則**一律引用真相來源**，場景不重定義、不硬編碼。

## 6. 健檢看板輸出契約（固定六欄位）

每檔持股 MUST 輸出以下六欄位，順序固定：

| # | 欄位 | 規格 |
|---|---|---|
| 1 | 健康燈號 | 綠／黃／紅（三選一） |
| 2 | 警訊清單 | 條列；無警訊時明示「無重大警訊」 |
| 3 | 建議行動 | 續抱／留意／減碼（三選一，long-only 持有語意，非做空） |
| 4 | 核心理由 | 條列 1–3 點 |
| 5 | 風險提示 | 非空文字 |
| 6 | data_gaps | 陣列，如實反映模組缺口，不得省略或偽裝為空 |

## 7. 燈號與建議行動判定

- **綠**（無重大警訊）→ 建議**續抱**
- **黃**（單一或次要警訊，如法人短賣、營收放緩）→ 建議**留意**
- **紅**（多項或重大警訊，如法人連續大賣 + 財報轉弱 + 處置）→ 建議**減碼**
- long-only：建議僅「續抱/留意/減碼」，**不含放空**；「減碼」= 退出，非做空。

## 8. 合規（紅線）

- **data_gaps 非空** → 看板 MUST 明示「以下健檢基於不完整資料」；為空不加此語。
- **每檔持股必附免責聲明**。
- **禁止**「保證」「必漲」「穩賺」等表述；燈號/建議僅為**參考資訊**。

## 9. 範例（健檢看板）

```
【2330 台積電】
健康燈號：🟡 黃
警訊清單：
  - 外資近 5 日連續淨賣超（佔量比 -4%）
  - 月營收 YoY 由 +22% 放緩至 +8%
建議行動：留意
核心理由：
  1. 法人短線調節，但大戶持股仍集中
  2. 營收成長放緩、非衰退
  3. 未觸及處置/漲跌停
風險提示：法人若續賣需留意；大盤系統性風險。
data_gaps：[]
⚠️ 免責：本健檢僅供研究參考，不構成投資建議，決策請自行判斷並自負風險。
```

data_gaps 非空時，看板頂端加註「以下健檢基於不完整資料」。

## 10. 使用方式

以自然語言提供持股即可，例如「幫我健檢持股 2330、2454、2317」。場景逐檔委派 `twstock-module` 取數、掃警訊、輸出上述看板。

## 11. 分層邊界與免責

- 本場景為 **L3 企業場景**：只做編排、警訊掃描、看板呈現；**取數一律委派 L2 `twstock-module`**。
- 所有輸出**不構成投資建議**。台股有漲跌停與流動性風險，AI 分析存在不確定性，投資決策請自行判斷並諮詢專業人士。
